# ScientificCalculator
Multi-mode calculator, does basic operations, scientic operations and grade calculation

After launching choose one of three modes
-first mode does basics operations (+, -, *, /)
-second mode does advanced operations (power, square root, percentage, trigonometry => sine, cosine, tangent)
-third mode collects a series of numbers and calculates the average grade, insert numbers like this (2;4;5,45)
 NB: if ',' doesn't work use '.' in double
